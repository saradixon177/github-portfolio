# Kpop Adventure Themed Game!

## Overview 
I am learning OOP and decided to take the typical game format and create a kpop themed adventure. To win the Ticketmaster War, you need to collect gold, photo cards, and pictures with your favorite idols to prove you deserve tickets to your favorite artist!

## Project Structure
```
├── README.md
├── adventure_game.py            # Python code that runs through the game
```
## Tools Used
- **Python**
- **VS Code**

## How to Reproduce
1. Download the adventure_game.py file
2. Run the file in your Python Interpreter of choice 
3. Enjoy dancing your way to the Ticketmaster War! I highly recommend putting on your favorite K-pop playlist
